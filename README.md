## Run Locally

Clone the project

```bash
  git clone https://github.com/praveen5959/notes-app
```

Go to the project directory

```bash
  cd notes-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
App will be accessible at http://localhost:3000/
